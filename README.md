### demo: https://route-manager.herokuapp.com/ (site may not be available)

For starting application:
- `cd ../route-manager`
- `yarn install`
- `yarn start`
- visit address in browser: `127.0.0.1`

If you wants to createa build:
- `cd ../route-manager`
- `yarn build`
- You may find build in `../route-manager/build`
